# PythonBasics_Lab1
Basic python syntax and arithmetic operations with input validation

### Basic task:
The program 
<li>takes two inputs and validates them using separate function </li>
<li>performs arithmetic operations: addition, subtraction, multiplication, division (with division by zero control)</li>
<li>prints the result</li>

### Individual task:
The program takes two other inputs, validates them and performs multiplication

### Additional functionality:
A custom function that validates input as a floating-point number and allows 3 input attempts
